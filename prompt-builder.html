Getting Started

  1. Search & Select Techniques - Use the search bar to find relevant techniques, then click to add them to your prompt.
  2. Fill In the Details - Add your role, task description, context, and desired output format in the compose tab.
  3. Review & Refine - Check the live preview to see your complete prompt and make adjustments as needed.
  4. Copy or Save - Copy your prompt to use immediately, or save it for later reuse.

Tips for Better Prompts

Be Specific: Clearly state what you want. Vague prompts lead to vague responses.

Provide Context: Include relevant background information that helps the AI understand your situation.

Use Examples: Show the AI what good output looks like by including sample inputs and outputs.

Define Format: Specify exactly how you want the response structured (lists, JSON, markdown, etc.).

Combine Techniques: Mix complementary techniques for more powerful results.

Recommended Combinations

Problem Solving: Chain-of-Thought + Self-Consistency for complex reasoning tasks.

Creative Tasks: Role Prompting + Few-Shot Learning for consistent creative output.

Code Generation: Structured CoT + Self-Correction for reliable, high-quality code.

Research: ReAct + Tree of Thoughts for thorough analysis and exploration.
